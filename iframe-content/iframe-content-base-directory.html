Hosted data for iframes

Folder Structure

Our iframe data organization.

The Full URL is built calling the "Base URL" + "File Directory Structure URI" +"HTML File Name"

This websites to embed the iframes can be found using the following syntax:

Base URL:

https://dc-policy-center.github.io/iframe-hosting-site-location/iframe-content/

File Directory Structure URI:

{!FOLDER NAME!}/{!CONTINUE FOLDER NAME AS NEEDED!}/

HTML File Name

Usually index.html, if not you will have to find the file on the github page. Do not forget the file extension .html
